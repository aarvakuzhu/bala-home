# bala-home

Chennai Home Floor Plan — Indian Vernacular Design  
**G+1 · 80×60 ft · 4800 sq ft · v7**

## Features

- Thinnai (raised front sit-out)
- Nadumuttam (open-sky internal courtyard)
- Sunken living room
- Dining hall — 21 ft double-height ceiling open to first floor
- Stair wraps around dining area
- Pooja room (east-facing)
- Open kitchen with half-wall partition
- Guest bedroom GF · Master + 2 child bedrooms FF
- Each child room with private bathroom
- Flex / Library / Soundproof room
- GF ceiling: 11 ft · FF ceiling: 10 ft

## Run locally

```bash
npm install
npm start
# → http://localhost:3000
```

## Deploy

Auto-deploys to Render on every push to `main` via `render.yaml`.
